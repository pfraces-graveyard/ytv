# ytv

play youtube videos from a remote controller

# install

    $ git clone http://github.com/paufraces/ytv
    # npm rebuild

# run

    $ node app

# routes

*   **/**         home
*   **/search**   search for videos
*   **/id/{id}**  video {id} detail with control functions
*   **/player**       video player
