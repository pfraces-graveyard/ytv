# ytv

reproducción de videos de youtube a través de control remoto

# instalar

    $ git clone http://github.com/paufraces/ytv
    # npm rebuild

# ejecutar

    $ node app

# rutas

todas las rutas parten de **http://localhost:3000**

*   **/**         página de inicio
*   **/rc**       control remoto
*   **/rc/{id}**  control remoto. detalle del video {id}
*   **/tv**       reproductor
